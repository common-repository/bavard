Adds the Bavard chatbot widget to your webpages.

== Description ==

This plugin acts as an interface to [Bavard AI](https://bavard.ai/), an external 3rd party service. [Bavard AI](https://bavard.ai/) is a software-as-a-service (SaaS) platform, providing software tools for building and deploying engaging and intelligent virtual agents at scale. Please click [here](https://bavard.ai/terms-of-service) to find Bavard's terms of use.

The plug-in will add the Bavard chatbot widget to your website. The widget loads a secure iframe window to help you engage and support your site's visitors with automated conversation.

== Installation ==

Before installing you must create and publish a virtual assistant using the Bavard core web app.

Next, simply go to "tools" -> "plugins" on your Wordpress site and search for the "Bavard" plugin.
Install and activate it, and then on the "Bavard Settings" page, fill in your assistant's name
and project api key. Finally, check the "enabled" checkbox, but make sure to leave the debug and development mode checkboxes UNCHECKED.

If you have any issues please email support@bavard.ai and we'll help you out right away.
